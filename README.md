## IntelliStocksAI
Challenge Fiap para a Plusoft

## Previsão de Vendas com Modelos de Regressão
Este projeto utiliza técnicas de aprendizado de máquina para prever vendas com base em dados históricos. Três modelos de regressão são aplicados e comparados: Árvore de Decisão, Random Forest e Gradient Boosting.

## Visão Geral
O objetivo deste projeto é prever as vendas futuras com base em dados históricos e identificar quais meses têm maior demanda prevista. Os modelos utilizados são:

Árvore de Decisão (DecisionTreeRegressor)
Random Forest (RandomForestRegressor)
Gradient Boosting (GradientBoostingRegressor)

## Dependências (Bibliotecas)
pandas
numpy
matplotlib
scikit-learn

## Resultados
O código imprime os três meses com maior demanda prevista para cada modelo e gera um gráfico comparativo. As previsões de vendas de cada modelo são visualizadas em comparação com os dados reais.
